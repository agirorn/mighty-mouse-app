mighty-mouse-app
================

HTML5 mobile mouse



Install
================
To install you need to hava have [ruby](http://www.ruby-lang.org) and [node.js](http://nodejs.org/).

```bash
  git clone https://github.com/agirorn/mighty-mouse-app.git
  cd mighty-mouse-app
  ./setup
```


Start
================
To install you need to hava have [ruby](http://www.ruby-lang.org) and [node.js](http://nodejs.org/).

```bash
  cd mighty-mouse-app
  ./start
```
